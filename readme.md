# Pirate Attack
Pirate Attack is a C++ game project developed from scratch without the use of any game engine. It is a tower attack strategy game where the player takes the role of an attacker while the opponent acts as a defender. The project focuses primarily on the implementation of core gameplay mechanics, game architecture, and AI behavior rather than on creating a fully polished commercial game.

## Info
- Genre: Tower Attack
- Theme: Pirates


## Used libraries
- lSDL2
- lSDL2_image
- lSDL2_ttf
- lSDL2_mixer
- lxml2
- lstdc++
